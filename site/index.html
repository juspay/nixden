devbox
GitHub Install Lima Start Script
Prebuilt NixOS on Lima

Local development VM, ready from a release image.

devbox starts a disposable NixOS machine without mounting your macOS home directory. Clone code inside the VM, keep file transfer explicit, and reconnect whenever you need a clean workstation.

  • Prebuilt GitHub release images
  • Practical flakes, direnv, gh, just, VS Code server
  • Contained only /tmp/lima-devbox is mounted

Start in one command

Install Lima first. On macOS, run brew install lima.

Reuses a running VM, starts a stopped VM, or creates one from the latest image.

latest release
curl -fsSL https://juspay.github.io/devbox/start | sh -

Typical workflow

1Open the VM

limactl shell --workdir=. devbox

2Clone inside devbox

mkdir -p ~/code && cd ~/code && git clone ...

3Use host VS Code

Remote-SSH config: ~/.lima/devbox/ssh.config, host lima-devbox.
